### Snippet

Quick start
* Run hugo server on root folder.

   ``` 
   hugo server 
   ````
* Create new blank post.

   ``` 
   hugo new posts/newblogpost.md
   ````
* Run **`` hugo ``** command for updating *public* folder, *public* folder is your static site.

   ``` 
   hugo 
   ````
